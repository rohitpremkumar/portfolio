Background

My name is Rohit Premkumar and I graduated from the New Jersey Institute of Technology with the intention of going to medical school one day. However, I had realized that I am more passionate about creating web applications and communicating data. I have taken a number of courses on data science covering topics including: wrangling, visualization, machine learning primarily with python. In addition, I have experience with full stack web development including this website which uses a Django back-end running on AWS Elastic Beanstalk and a Boostrap front-end. Investing in equities is a hobby that I have used to build an impressive portfolio and gain valuable insight into the financial world.

My overarching goal is to work in the FinTech sector, developing applications that help high-profile traders and the average-joe(like me) make more intelligent investment decisions.
